## 분석

### 잘된 점
-전체 mAP50 = 0.915, baseline 치고 매우 높음.

클래스 대부분이 mAP50 = 0.99 이상으로 잘 학습됨.

데이터셋이 작지만 YOLOv8n이 빠르게 수렴함.

클래스 imbalance 문제에도 불구하고 성능 저하 적음.

### 문제점

-일부 클래스(예: 에스원엠프정 10mg/병)에서 낮은 성능(mAP50 = 0.497).

-Data augmentation이 기본값이라 다양성이 부족할 수 있음.


-GPU(RTX 3050 4GB)의 VRAM이 적어 batch size 제약 가능성 있음.

### 개선 방향

-부족한 클래스에 대해 이미지 증강을 추가로 적용
(color jitter, mosaic 강도 조절, copy-paste augmentation)

-Model capacity를 올려 YOLOv8s 실험 진행

-Epoch 증가(50 → 100) 실험

-Batch size -> 32로 키워 gradient 안정성 확보, 만약 GPU로 batch size 제약 있으면 colab pro에서 실험

-Learning rate scheduler 조정 (cosine → one-cycle 등)
