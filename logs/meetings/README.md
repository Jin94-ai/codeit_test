# 회의록 작성 가이드

팀 미팅 후 바로 작성하세요.

---

## 파일명 규칙

```
YYYY-MM-DD_제목.md
```

**예시**:
- `2025-12-05_첫미팅.md`
- `2025-12-13_Week1회고.md`

---

## 작성 팁

### 안건
- 미팅 전에 미리 정리
- 우선순위 순서로 나열

### 논의 내용
- 핵심만 간단히
- 누가 어떤 의견을 냈는지 기록

### 결정 사항
- 명확하고 구체적으로
- 체크박스로 완료 여부 추적

### 액션 아이템
- 담당자, 작업, 마감일 필수
- 상태: ⏳(진행중) / ✅(완료) / ⚠️(지연)

---

## 예시

```markdown
# 팀 미팅 - 2025-12-05

## 회의 정보
- **날짜**: 2025-12-05
- **시간**: 19:00 ~ 20:00 (총 60분)
- **참석자**: 이진석, 김민우, 박수진, 이하늘, 최지우
- **불참자**: 없음

## 안건
1. 자기소개 및 팀 빌딩
2. 기술 스택 결정
3. 역할 분배
4. 일정 및 협업 규칙 합의

## 논의 내용

### 1. 자기소개 및 팀 빌딩
- 각자 5분씩 자기소개 완료
- 강점과 관심사 공유
- 진석: 프로젝트 관리 경험, Git 숙련
- 민우: 데이터 분석 경험, Pandas 능숙
- 수진: PyTorch 사용 경험, 컴퓨터 비전 관심

### 2. 기술 스택 결정
- Object Detection 모델: YOLOv8 vs Faster R-CNN
  - 민우: 시간 제약 고려해 YOLOv8 제안
  - 수진: 성능 우선 Faster R-CNN 의견
  - 결론: YOLOv8로 베이스라인, 시간 여유 있으면 Faster R-CNN 실험
- 프레임워크: PyTorch (만장일치)
- 실험 추적: Weights & Biases (만장일치)

### 3. 역할 분배
- TEAM_ROLES.md 기반으로 논의
- 각자 원하는 역할 지원 후 조율
- 강점 고려하여 최종 확정

### 4. 일정 및 협업 규칙
- 일일 스탠드업: 매일 오전 10시 (Discord)
- 주간 회고: 매주 금요일 오후 7시
- 협업 일지: 매일 작성 필수
```
