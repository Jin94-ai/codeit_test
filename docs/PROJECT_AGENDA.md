## 🔍 경쟁사 분석

### 해외 시장

| 앱 | 사진 인식 | 상호작용 체크 | 한국 약 | 비고 |
|:---|:--------:|:------------:|:------:|------|
| Drugs.com | ✅ | ✅ | ❌ | 미국 최대, 24,000+ 약물 DB |
| Smart Pill ID | ✅ AI | ✅ | ❌ | AI 사진 인식 |
| Pill Identifier | ✅ AI | ✅ | ❌ | 즉시 인식 |

> **한계**: 해외 앱은 **한국 약 미지원**

### 국내 시장

| 앱/서비스 | 사진 인식 | 상호작용 체크 | 비고 |
|:----------|:--------:|:------------:|------|
| 식약처 의약품 식별 | ❌ | ❌ | **수동 검색** (색상, 모양 선택) |
| PillEye (필아이) | ❌ | ❌ | 알약 **세기** 전용 (약사용) |
| Medisafe | ❌ | 🔺 | 복용 알림 앱, 인식 없음 |

### 시장 기회

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   🇺🇸 해외: 사진 인식 + 상호작용 체크 앱 다수 존재                    │
│         → 하지만 한국 약 DB 없음                                     │
│                                                                     │
│   🇰🇷 한국: 우리가 원하는 기능을 가진 앱 거의 없음                    │
│         → 식약처 수동 검색만 존재                                    │
│         → AI 사진 인식 + 상호작용 = 블루오션                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Health Eat 차별점

| 기존 서비스 | 문제점 | Health Eat 솔루션 |
|------------|--------|------------------|
| 식약처 검색 | 수동 입력 (색상, 모양, 각인) | **사진 한 장으로 자동 인식** |
| 해외 앱 | 한국 약 미지원 | **한국 약 DB** |
| 해외 앱 | 영어만 지원 | **한국어 지원** |
| 기존 앱들 | 상호작용 별도 앱 | **인식 + 상호작용 통합** |

## 👥 타겟 사용자

### 우선순위

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   1️⃣  일반 사용자 + 여행자                                 [1순위]  │
│       ─────────────────────                                        │
│       • 약 이름/효능 모르는 일반인                                  │
│       • 해외 여행 중 현지 약 구매하는 여행자                         │
│       • 한국 방문 외국인                                            │
│                                                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   2️⃣  요양원 / 재가요양                                    [2순위]  │
│       ─────────────────────                                        │
│       • 다약제 복용 어르신 관리                                     │
│       • 비전문가 요양보호사의 약 관리 보조                           │
│       • B2B SaaS 수익 모델                                          │
│                                                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   3️⃣  약국                                                 [3순위]  │
│       ─────────────────────                                        │
│       • 환자가 알약만 가져왔을 때 빠른 확인용                        │
│       • 보조 도구 (전문가이므로 낮은 우선순위)                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### Pain Point

| 타겟 | 문제 | 솔루션 |
|:----:|------|--------|
| 💊 **일반인** | 약 이름/복용법 모름 | 사진 → 즉시 정보 |
| ✈️ **여행자** | 해외 약 성분 확인 불가 | 성분 매칭 엔진 |
| 🏥 **요양원** | 다약제 상호작용 관리 어려움 | 자동 상호작용 체크 |
| ⚠️ **공통** | 잘못된 복용 위험 | 안전 경고 시스템 |

### 사용 시나리오

```
📍 어디서든 → 📸 사진 촬영 → 💡 즉시 정보 확인

🌏 해외 약국 → 📸 현지 약 촬영 → 🇰🇷 한국 동일 성분 약 매칭
```

## ✅ 기능 우선순위

### Must Have (필수)

```
┌─────────────────────────────────────────────────────────────────────┐
│  1️⃣  알약 정확 검출           confidence ≥ 0.9                      │
├─────────────────────────────────────────────────────────────────────┤
│  2️⃣  불확실성 처리 (핵심!)    confidence < 0.9 → "다시 촬영" 안내   │
│                              confidence < 0.7 → "인식 불가" 표시    │
├─────────────────────────────────────────────────────────────────────┤
│  3️⃣  알약 효능 정보           해당 약의 효능/효과 표시               │
├─────────────────────────────────────────────────────────────────────┤
│  4️⃣  복용 주의점              복용 시 주의사항 안내                  │
├─────────────────────────────────────────────────────────────────────┤
│  5️⃣  상호작용 체크            함께 복용 시 주의해야 할 조합          │
└─────────────────────────────────────────────────────────────────────┘
```

### 불확실성 처리 (Uncertainty Handling)

```
                      사진 입력
                          │
                          ▼
                  ┌───────────────┐
                  │   AI 분석     │
                  └───────┬───────┘
                          │
              ┌───────────┼───────────┐
              │           │           │
              ▼           ▼           ▼
        conf ≥ 0.9   0.7~0.9     conf < 0.7
              │           │           │
              ▼           ▼           ▼
      ┌───────────┐ ┌───────────┐ ┌───────────┐
      │ ✅ 결과   │ │ ⚠️ 재촬영  │ │ ❌ 인식   │
      │    표시   │ │    요청   │ │    불가   │
      └───────────┘ └───────────┘ └───────────┘
                          │           │
                          ▼           ▼
                    "더 밝은 곳에서"  "약사에게
                    "정면으로"       문의하세요"
```

> **의료 AI의 철칙: "틀리는 것보다 모른다고 하는 게 100배 낫다"**

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   📱 Mobile App                    ☁️ Cloud Server                   │

│   ┌─────────────┐                 ┌─────────────────────────────┐  │
│   │  Android    │    HTTPS API    │        API Server           │  │
│   │    App      │ ──────────────▶ │        (FastAPI)            │  │
│   │  (~20MB)    │                 └──────────┬──────────────────┘  │
│   └─────────────┘                            │                      │
│                                              ▼                      │
│   • Kotlin                        ┌─────────────────────────────┐  │
│   • Jetpack Compose               │        AI Server            │  │
│   • Retrofit (HTTP)               │   YOLO11s (PyTorch + GPU)   │  │
│   • CameraX                       └──────────┬──────────────────┘  │
│                                              │                      │
│                                              ▼                      │
│                                   ┌─────────────────────────────┐  │
│                                   │       PostgreSQL            │  │
│                                   │       (약물 정보 DB)         │  │
│                                   └─────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
